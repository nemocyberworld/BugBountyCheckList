1. Quick WHOIS & DNS Commands (Single Target)

Replace example.com with your target domain.

# WHOIS Lookup
whois example.com

# DNS Queries (A, MX, TXT, ANY)
dig example.com any +noall +answer
dig mx example.com
dig txt example.com

# Nameserver Info
dig ns example.com
nslookup -type=ns example.com

2. Why WHOIS & DNS Recon Matters

  • Identify domain ownership and registration patterns.
  • Reveal mail servers, nameservers, and subdomains.
  • Detect possible misconfigurations or takeover risks.

3. Simple Automation Script (Single Target)

#!/usr/bin/env python3
import os, subprocess, sys

if len(sys.argv) != 2:
    print("Usage: python3 recon.py ")
    sys.exit(1)

domain = sys.argv[1]
commands = {
    "WHOIS Info": f"whois {domain}",
    "DNS ANY": f"dig {domain} any +noall +answer",
    "DNS MX": f"dig mx {domain}",
    "DNS TXT": f"dig txt {domain}",
    "NSLookup ANY": f"nslookup -type=any {domain}"
}

for title, cmd in commands.items():
    print(f"\n[+] {title}")
    os.system(cmd)
