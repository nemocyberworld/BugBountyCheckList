This guide summarizes popular subdomain enumeration tools for a single domain. It covers passive enumeration, brute-force/permutation, aggregating results, resolving live subdomains, takeover checks, and screenshots.

1. Passive Enumeration Tools (Single Domain)

Amass

Passive enumeration using multiple data sources and APIs.

amass enum -passive -d <domain> -o amass.txt
import re

def extract_records(filename):
    pattern = re.compile(r'--> ([^ ]+) -->')
    domain_pattern = re.compile(r'\b(?:[a-zA-Z0-9-]+\.)+[a-zA-Z]{2,}\b')

    records, subdomains = {}, set()
    with open(filename, 'r') as f:
        for line in f:
            line = line.rstrip('\n')
            m = pattern.search(line)
            if m:
                typ = m.group(1)
                records.setdefault(typ, []).append(line)
            for d in domain_pattern.findall(line):
                subdomains.add(d)

    for typ, lines in records.items():
        with open(f"{typ}.txt", 'w') as out:
            out.write("\n".join(lines) + "\n")
        print(f"Wrote {len(lines)} lines to {typ}.txt")

    if subdomains:
        with open("subdomains.txt", 'w') as fsub:
            fsub.write("\n".join(sorted(subdomains)) + "\n")
        print(f"Wrote {len(subdomains)} unique subdomains to subdomains.txt")

if __name__ == "__main__":
    extract_records('amass.txt')

Subfinder

Fast passive subdomain discovery.

subfinder -d <domain> -o subfinder.txt

Assetfinder

Find subdomains using public sources.

assetfinder --subs-only <domain> > assetfinder.txt

Sublist3r

Enumerate subdomains via search engines and APIs.

sublist3r -d <domain> -o sublist3r.txt

DNSRecon üîç

Advanced DNS enumeration and reconnaissance.

dnsrecon -d <domain> -n 8.8.8.8 -j dnsrecon.json
Standard:
  dnsrecon -d example.com
AXFR (if misconfigured):
  dnsrecon -d example.com -a
Bruteforce:
  dnsrecon -d example.com -D wordlist.txt -t brt
Zone walking:
  dnsrecon -d example.com -t zonewalk

TheHarvester üîç

Harvest subdomains and emails from public sources.

theharvester -d <domain> -b all -f theharvester.xml
Basic:
  theharvester -d example.com -b google
Multiple engines + HTML:
  theharvester -d example.com -b google,bing,linkedin -f report.html
Limit results:
  theharvester -d example.com -b bing -l 100

2. Aggregate & Deduplicate (Single Domain)

Combine results and remove duplicates:

cat results/example.com/*.txt | sort -u > results/example.com/all_subs.txt

3. Resolve Live Subdomains (Single Domain)

Filter which subdomains respond over HTTP/HTTPS:

httpx -l results/example.com/all_subs.txt -threads 50 -o results/example.com/live_subs.txt

4. Subdomain Takeover Checks (Single Domain)

subjack -w results/example.com/live_subs.txt -t 50 -timeout 30 -ssl -v -o results/example.com/subjack.txt
subover -l results/example.com/live_subs.txt -o results/example.com/subover.txt
second-order -d example.com -l results/example.com/live_subs.txt -o results/example.com/second_order.txt

5. Screenshot Live Subdomains (Single Domain)

Capture screenshots of live subdomains:

gowitness file -f results/example.com/live_subs.txt -P results/example.com/screenshots --threads 50

6. Tools in Kali Linux & BlackArch with Single-Target Commands

Note: Many of these are available in both distros; install via the package manager if missing. Replace <domain> with your target domain.

Kali Linux (common subdomain/DNS tools)

  • amass ‚Äî Advanced passive/active subdomain enumeration framework.
    amass enum -passive -d <domain> -o amass.txt
  • subfinder ‚Äî Fast passive subdomain discovery from public sources.
    subfinder -d <domain> -o subfinder.txt
  • assetfinder ‚Äî Finds subdomains and related assets using public data.
    assetfinder --subs-only <domain> > assetfinder.txt
  • sublist3r ‚Äî Enumerates subdomains using multiple search engines and APIs.
    sublist3r -d <domain> -o sublist3r.txt
  • dnsrecon ‚Äî Advanced DNS enumeration tool with multiple modes.
    dnsrecon -d <domain> -n 8.8.8.8 -j dnsrecon.json
  • dnsenum ‚Äî Multi-threaded Perl script for DNS information gathering.
    dnsenum <domain> --dnsserver 8.8.8.8 --enum -o dnsenum.xml
  • fierce ‚Äî DNS reconnaissance tool to locate non-contiguous IP space and hostnames.
    fierce --domain <domain> --wide
  • findomain ‚Äî Fast subdomain discovery using multiple APIs.
    findomain -t <domain> -u findomain.txt
  • knockpy ‚Äî Python tool for subdomain enumeration via wordlists.
    knockpy <domain> -o knockpy/
  • dnsx ‚Äî High-performance DNS resolver and subdomain status checker.
    dnsx -d <domain> -r 1.1.1.1 -o dnsx.txt
  • massdns ‚Äî Massively parallel high-speed DNS resolver.
    massdns -r resolvers.txt -t A -o S -w massdns.txt wordlist.txt
  • puredns ‚Äî Fast and accurate DNS brute-forcer with wildcard filtering.
    puredns bruteforce words.txt <domain> -r resolvers.txt -w puredns.txt
  • altdns ‚Äî Generates permutations, alterations, and mutations of subdomains.
    altdns -i subdomains.txt -o altdns_out.txt -w words.txt -r -s altdns_final.txt
  • gotator ‚Äî Permutation generator for wordlist-based subdomain enumeration.
    gotator -sub subdomains.txt -perm permutations.txt -depth 1 -adv -silent > gotator.txt
  • dnsgen ‚Äî Generates domain name permutations from existing subdomains.
    dnsgen subdomains.txt | tee dnsgen.txt
  • httpx ‚Äî Fast and multi-purpose HTTP toolkit for probing live hosts.
    httpx -l all_subs.txt -o live_subs.txt
  • subjack ‚Äî Subdomain takeover tool for detecting vulnerable services.
    subjack -w live_subs.txt -t 50 -ssl -v -o subjack.txt
  • subover ‚Äî Tool to detect subdomain takeovers using a large fingerprint database.
    subover -l live_subs.txt -o subover.txt
  • aquatone ‚Äî Visual inspection tool for website reconnaissance.
    cat live_subs.txt | aquatone -out aquatone/
  • gowitness ‚Äî Screenshot utility for capturing web pages.
    gowitness file -f live_subs.txt -P screenshots

Summary

  • Start with passive sources (Amass/Subfinder/Assetfinder).
  • Expand via permutations (altdns/dnsgen/gotator) and brute-force (massdns/puredns).
  • Aggregate & dedupe, then resolve (dnsx/httpx).
  • Check takeover (subjack/subover) and capture screenshots (gowitness/aquatone).
