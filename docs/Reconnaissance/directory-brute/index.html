Feroxbuster

A fast and simple directory/file bruteforcer written in Rust.

feroxbuster -u http://<target> -w wordlist.txt -r -x php,sql,bak,conf
#!/bin/bash
# Feroxbuster single-target helper

read -p "Target URL (e.g., https://example.com): " TARGET_URL
read -p "Path to wordlist: " WORDLIST

EXTS=$(echo "php,asp,aspx,jsp,cgi,pl,py,rb,go,conf,cfg,ini,yaml,yml,json,xml,env,bak,old,backup,zip,tar,tar.gz,rar,7z,txt,log,md,csv,sql,db,sqlite,sqlite3,html,htm,js,css,map,pdf,doc,docx,xls,xlsx,ppt,pptx,pem,crt,key" | tr -d ' ')
echo -e "\nüöÄ Scanning $TARGET_URL ..."
feroxbuster -u "$TARGET_URL" \
  -w "$WORDLIST" \
  -x "$EXTS" \
  -r -t 100 --depth 3 \
  --status-codes 200,204,301,302,307,401,403 \
  -o "ferox-output.txt"

echo -e "\n‚úÖ Done. Saved: ferox-output.txt"

Other Popular Single-Target Commands (Quick Reference)

ffuf

Blazing-fast web fuzzer for content discovery and parameter fuzzing. 
ffuf -u http://<target>/FUZZ -w wordlist.txt -mc 200,204,301,302,307,401,403 -of csv -o ffuf.csv

Gobuster (dir mode)

Directory and file brute-forcer with clean output and extension handling. 
gobuster dir -u http://<target> -w wordlist.txt -x php,txt,bak -t 50 -o gobuster.txt

dirsearch

Python-based brute-forcer with smart auto-tuning and recursion. 
dirsearch -u http://<target> -w wordlist.txt -e php,txt,bak -t 50 -o dirsearch.txt

dirb

Classic directory brute-forcer with bundled wordlists. 
dirb http://<target> /usr/share/dirb/wordlists/common.txt -o dirb.txt

wfuzz

Highly flexible web fuzzer for paths, params, headers, and more. 
wfuzz -c -z file,wordlist.txt --hc 404 http://<target>/FUZZ

nikto (baseline checks)

Web server scanner (not a brute-forcer) ‚Äî good for quick misconfig checks. 
nikto -host http://<target> -output nikto.txt

Tools in Kali Linux & BlackArch for Directory Bruteforcing (Single Target)

Notes: These are the most-used tools for directory/content discovery. Many are present in both distros. Replace <target> with your host.

Kali Linux

    feroxbuster -u http://<target> -w wordlist.txt -r
